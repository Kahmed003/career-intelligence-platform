# Asset

Definition

A long-term capability or resource that compounds over time and increases career value.

Examples

- Programming
- Research Profile
- Leadership
- Professional Network

Related Terms

- Goal
- Project
- Skill

Appears In

- PRD
- Domain Model

# Opportunity

Definition

Any external event, position, program, or relationship capable of advancing one or more long-term goals.


# Knowledge

Definition

Reusable understanding created from one or more pieces of evidence.

Not to be confused with Documents or Notes.
